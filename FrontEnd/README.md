
# Frontend - Proyecto Banco ISTLC

Este frontend es una SPA desarrollada en React para simular un sistema bancario moderno, con login, registro de clientes, cajero automático y panel de usuario.

## Tecnologías principales
- React (Vite)
- React Router DOM
- CSS modular

## Funcionalidades
- **Login:** Acceso seguro con validación de credenciales.
- **Registro de cliente:** Formulario para nuevos usuarios (requiere cédula y correo).
- **Cajero automático:** Depósito y retiro con validación y mensajes claros.
- **Panel de usuario:** Consulta de saldo, historial y transferencias.

## Instalación y ejecución

1. Instala dependencias:
	```bash
	npm install
	```
2. Inicia la aplicación:
	```bash
	npm run dev
	```

## Notas
- El frontend se conecta al backend en `http://localhost:2025`.
- Personaliza los estilos en los archivos `.css` de `src/`.
